# Mr Ahmed – Professional OSCE Virtual Patient App

This version includes:
- Patient portrait in the header
- Natural rapport/small-talk responses
- Clinical document viewer
- OSCE assessment scoring
- Clinical reasoning feedback
- SOAP note export/print-to-PDF
- Student performance dashboard
- Lecturer/admin mode
- PWA install settings and offline support

## To use locally
Open `index.html`.

## To publish as an app
Upload the whole folder to GitHub Pages or Netlify. Then students can open the web link and choose **Add to Home Screen** / **Install App**.

## Important
Keep the `assets` folder, `manifest.json`, and `service-worker.js` with `index.html`.
